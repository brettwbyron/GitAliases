# objects/flex.scss

## Flexbox Layout Classes

[CSS Flexbox Guide from CSS-Tricks](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)

The flex classes use abbreviations, see below to understand.

```
// -----------------------------------------------------------------------------
// =Flex classes
// -----------------------------------------------------------------------------
// m = main axis
// w = wrap or space along cross axis
// c = cross axis alignment
// self = align-self
```

## Example HTML

Use the classes in combination to control how the child elements align. Using classes for media-queries like "-sm, -md, -lg, -xl" will allow for greater control of layouts.

```
<div class="flex-col flex-row-md flex-center">
    <div> </div>
    <div> </div>
    <div> </div>
</div>
```
